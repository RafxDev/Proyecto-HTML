# 📖 Acerca del proyecto

Este proyecto fue desarrollado como material de apoyo para enseñar **HTML y CSS** a un grupo de jóvenes programadores.

En lugar de construir una página sobre un tema convencional, decidí utilizar el **Live Coding**, una disciplina que une programación, música y creatividad, permitiendo que los estudiantes aprendieran desarrollo web mientras descubrían una de las ramas más interesantes de la informática artística.

El sitio fue realizado únicamente con **HTML5** y **CSS3**, manteniendo el proyecto sencillo para que cualquier persona que esté comenzando a programar pueda comprender su estructura y funcionamiento.

---

# 💻 ¿Qué es el Live Coding?

El **Live Coding** consiste en crear música, visuales o experiencias digitales escribiendo código en tiempo real frente a una audiencia.

Cada línea de código modifica inmediatamente la composición, convirtiendo al lenguaje de programación en un verdadero instrumento musical.

Actualmente existen herramientas como:

- 🎹 Strudel
- 🎼 TidalCycles
- 🎧 Sonic Pi
- 🎛️ SuperCollider
- 🐍 FoxDot
- 🎨 Hydra

que permiten explorar esta forma de expresión artística.
