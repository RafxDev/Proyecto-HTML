Hechos Que Forjaron El Live Coding

Año Persona Aporte
1957 Max Mathews Desarrolla MUSIC I en Bell Labs, considerado el primer lenguaje para síntesis digital de sonido.
1963 Max Mathews Presenta MUSIC IV, ampliamente utilizado en investigación musical.
1968 Douglas Engelbart Demuestra nuevas formas de interacción entre humanos y computadoras, inspirando herramientas creativas en tiempo real.
1969 John Chowning Descubre la síntesis FM, utilizada posteriormente en numerosos sintetizadores digitales.
1970 Barry Vercoe Inicia el desarrollo de MUSIC 11, base de futuros lenguajes musicales.
1985 Miller Puckette Comienza el desarrollo de Max, entorno de programación visual para música interactiva.
1991 Miller Puckette Crea Pure Data (Pd), software libre ampliamente usado para arte interactivo y Live Coding.
1996 James McCartney Se publica SuperCollider, uno de los sistemas más importantes para síntesis y Live Coding.
2001 Adrian Ward Populariza el término "Live Coding" mediante presentaciones donde programa arte en vivo.
2003 Alex McLean Comienza el desarrollo de TidalCycles y promueve el Live Coding musical.
2004 TOPLAP Se funda la organización TOPLAP (Temporary Organisation for the Promotion of Live Algorithm Programming).
2005 Nick Collins Publica investigaciones fundamentales sobre Live Coding y música algorítmica.
2006 Slub Se consolida como uno de los primeros grupos de Live Coding musical.
2008 Algorave Comienzan los primeros eventos que mezclan música electrónica y programación en vivo.
2011 Sam Aaron Inicia el desarrollo de Sonic Pi con fines educativos.
2013 Sonic Pi Se publica oficialmente y comienza a utilizarse en escuelas y universidades.
2014 Algorave El movimiento se expande por Europa, Asia y América.
2016 FoxDot Gana gran popularidad entre la comunidad Python.
2017 Olivia Jack Presenta Hydra, herramienta para Live Coding visual.
2020 Strudel Se inicia el desarrollo de Strudel como alternativa web inspirada en TidalCycles.
2023 Strudel Se convierte en una de las plataformas de Live Coding más utilizadas desde el navegador.
Actualidad Comunidad internacional El Live Coding se utiliza en conciertos, festivales, investigación, educación y arte digital.
